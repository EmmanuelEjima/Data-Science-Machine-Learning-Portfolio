# 🛒 Supermarket Sales Prediction

## Overview

This project develops a machine learning regression model to predict supermarket sales using historical transaction data.

## Objective

- Predict total sales per transaction.
- Identify factors affecting sales.
- Generate business insights for inventory management and decision-making.

## Dataset

Public Supermarket Sales Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Machine Learning

- Data Cleaning
- Feature Engineering
- Regression Models
- Model Evaluation
